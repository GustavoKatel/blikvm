## Features

- Support  Raspberry Pi 4B and CM4, PCIe card.
- **Video capture** (HDMI/DVI/VGA)
- **Keyboard forwarding**
- **Mouse forwarding**
- **ATX** Control the server power using ATX functions
- **Fullscreen mode**
- Access via **Web UI**
- Support **multi language** switching
- **Health monitoring** of the Pi
- PWM **FAN** controller
- Support **PoE**
- **Serial** console port
- **I2C** display connector
- **Real Time Clock (RTC)**
