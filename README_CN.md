## 功能特点

- **视频捕获**（HDMI/DVI/VGA）
- **键盘转发**
- **鼠标转发**
- **虚拟U盘（重装系统）**
- **ATX** 使用 ATX 功能控制服务器电源
- **全屏模式**
- 通过 **Web UI** 访问
- 支持 **多语言** 切换
- 对 Pi 进行 **健康监控**
- PWM **风扇** 控制器
- 支持 **PoE**
- **串口** 控制台端口
- **I2C** 显示器连接器
- **实时时钟（RTC）**
